# 🔐 Task Manager

A simple, clean and responsive **Task Manager Web Application** built using **HTML, CSS, and JavaScript**.  
This application helps users manage their daily tasks efficiently with an easy-to-use interface.

---

## 🚀 Live Demo

Live Website:

👉 https://task-manager-vert-ten-50.vercel.app/

---

## ✨ Features

➕ Add new tasks  
🗑️ Delete tasks  
✔️ Mark tasks as completed  
💾 Tasks saved using localStorage (if implemented)  
📱 Fully responsive design  
🎨 Clean and modern UI  
⚡ Lightweight and fast performance  

---

## ⚙️ How to Run the Project

### Option 1: Open Directly
- Download or clone the repository
- Open `index.html` in your browser

### Option 2: Using VS Code
- Open project in VS Code  
- Install Live Server extension  
- Right click `index.html` → Open with Live Server  

---

## 💡 How It Works

- User adds a task using input field  
- Task gets displayed in the list  
- User can mark task as completed or delete it  
- Data may persist using browser storage
